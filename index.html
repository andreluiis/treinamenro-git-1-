Perguntas.

1- Quais os comandos para configurar usuário e email no git?

R: Os comandos para para configuração do git são: git config --global user.name "usuário" e git config -- global user.email "email-do-usuário".

2- Qual a diferença entre git e github?

R: github é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Git e um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.
