Perguntas.

1- Quais os comandos para configurar usuário e email no git?


R: Os comandos para para configuração do git são: git config --global user.name "usuário" e git config -- global user.email "email-do-usuário".

2- Qual a diferença entre git e github?


R: github é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Git e um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.

3- O que faz o comando commit?


R:Em ciência da computação e gerenciamento de dados, uma consolidação é tornar permanente um conjunto de alterações provisórias, marcando o final de uma transação e fornecendo Durabilidade às transações ACID.

O que faz o comando push?


R: O comando push transfere commits do repositório local a um repositório remoto.

5- O que faz o comando pull?


R: O comando git pull é usado para buscar e baixar conteúdo de repositórios remotos e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais.
